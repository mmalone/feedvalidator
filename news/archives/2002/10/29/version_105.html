Version 1.0.5

11:04 PM on Tuesday, October 29, 2002

Version 1.0.5 is out. Fixes:

  • Added missing documentation pages: ContainsEmbed, ContainsObject, ContainsMeta, ContainsScript.
  • Added new warning: ContainsRelRef, for description elements that contain HTML that contains links or images that contain relative URL references.
  • Fixed bug detecting script elements in entity-encoded description.
