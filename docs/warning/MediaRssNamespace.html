Message

The prefix "media" generally is associated with the namespace "http://search.yahoo.com/mrss/"

Explanation

The namespace defined by the Media RSS specification changes over time. This appears to be a mistake, and the latest word from the specification’s primary editor is that the version of the namespace which includes a trailing slash is the correct one, despite whatever the specification may say at any given point in time.

Here are a few links to prior discussions on the topic:

  • David Hall
  • James Holderness

Solution

As this is an area of continuing confusion, consumers would be well adivised to accept both forms of the namespace URI.

Producers of feeds should seek advice from the rss-media Yahoo! group as to which version is prefered at the moment.
